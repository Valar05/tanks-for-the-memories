## How To Use This Index

1. Start from `CODEX_STARTS_HERE.md` and the newest prompt.

3. Open the per-attempt note before copying any code.
4. Treat high IoU source-faithful attempts as warning signs, not candidates.
5. Treat topology-passing slab/prism attempts as warning signs, not candidates.
