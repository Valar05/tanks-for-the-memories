# AVRE Radio Command Organ

The current playable slice is a Royal Engineers Churchill AVRE breach operation.

The player is not a driver, gunner, radio operator, or detached tactical cursor. The player is an engineer whose practical hands are other people: driver, gunner, wireless operator, infantry liaison, and sapper team.

The command organ is bounded:

- normalize one typed or speech-recognition utterance
- match only registered actions
- bind the capable recipient
- choose the correct internal or external route
- validate required operational meaning
- dispatch one typed command or return `null`

`null` means radio silence. Incomprehensible, ambiguous, incomplete, or garbage utterances cause no action, no acknowledgement, no invented correction, and no relationship mutation. Speaking may still spend time.

Physically meaningful commands that are impossible in the current world state are different from garbage. They may route to the correct recipient, who may report the physical obstruction tersely. The compiler still does not invent missing meaning.

Cartesia audio is static content. Authored text is canonical and must remain complete without audio. The browser never receives a Cartesia key.
