# Archival Memory Visual Doctrine

## Core Rule

Do not make Tanks for the Memories literally black and white.

Use archival tank footage as emotional reference, not as a permanent filter.

## Target Feeling

The game should feel remembered, not merely observed.

Old black-and-white footage suggests:

- grain
- partial evidence
- haze
- silhouettes
- incomplete context
- uncertain identification
- reconstructed events

Use those qualities to support the information-pressure loop.

## Normal Gameplay

Normal gameplay stays in muted color:

- muddy greens
- washed-out browns
- gray skies
- wet stone
- dull steel
- smoke-stained air

Avoid heroic saturation and toy-like color.

## Memory / Evidence Moments

Use short archival-style effects only for important information or memory events:

- desaturation pulse
- film grain pulse
- scope vignette
- exposure flicker
- frame jitter
- smoke occlusion
- low-contrast haze
- radio static swell

These effects should never make play unreadable.

## Design Rule

Archival styling means evidence, uncertainty, or reconstruction.

It is not nostalgia.

It is not a documentary filter.

## Relationship To Camera Doctrine

The battlefield is a collection of human observations.

The camera should show fragments that matter, not omniscient truth.

## Acceptance Criteria

This doctrine works when:

- normal play remains readable in muted color
- special moments briefly feel archival
- visual degradation communicates uncertainty rather than decoration
- the player feels they are reconstructing events from evidence

## Reusable Quote

Do not make the game black and white. Make it feel like historical memory under pressure.
